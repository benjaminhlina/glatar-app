# Source Table Sidebar

Provides the sidebar for the source table pane.

## Usage

``` r
source_sidebar_ui(id)

source_sidebar_server(id, main_input, source_sidebar_vals)
```

## Arguments

- id:

  the shiny namespace id name (i.e., `"source_sidebar"`).

- main_input:

  the shiny input from the main server

## Details

`source_sidebar_ui()` provides the summary table sidebar user interface.

`source_sidebar_server()` provides the summary table sidebar server.
