# Tab Register Modal

This function produces the registration modal on the login modal when
the registration button is slected

## Usage

``` r
tab_register_modal(failed = FALSE, success = FALSE)
```

## Arguments

- failed:

  a logical value that when supplied prodocues warning when not all
  fields are filled out. Default is `FALSE`.

- success:

  a logical value that when supplied prodocues a success messsage.
  Default is `FALSE`.
