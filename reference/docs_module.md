# Documentation User Interface

Provides the User Interface (UI) for the documentation tab

Provides the server for the documentation tab

## Usage

``` r
docs_ui(id)

docs_server(id)
```

## Arguments

- id:

  the shiny namespace id name (i.e., `"docs"`).

## Value

`docs_ui()` a shiny UI.

`docs_server()` a shiny module server.
