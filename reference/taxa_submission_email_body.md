# Taxa Submission Email Body

This function drafts the text body the taxa submission email that will
be sent once the submit to manager has been hit.

## Usage

``` r
taxa_submission_email_body(n_rows, submitted_by)
```

## Arguments

- n_rows:

  the number of rows being submitted

- submitted_by:

  the email of the submitter

## Value

HTML of the email structure to be sent.
