# How to Use Tab

Provides the How to Use tab in the app.

## Usage

``` r
how_to_ui(id)

how_to_server(id, parent_session)
```

## Arguments

- id:

  the shiny namespace id name (i.e., `"how_to_use"`).

- parent_session:

  a shiny server session.

## Details

`how_to_ui()` a shiny UI.

`how_to_server()` a shiny module server.
