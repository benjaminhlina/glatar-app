# About User Interface

Provides the User Interface (UI) for the about tab

Provides the server for the about tab

## Usage

``` r
about_ui(id)

about_server(id)
```

## Arguments

- id:

  the shiny namespace id name (i.e., `"about"`).

## Value

`about_ui()` a shiny UI.

`about_server()` a shiny module server.
