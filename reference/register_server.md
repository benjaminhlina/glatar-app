# Registration Server

Porvides the server side for the
[`register_ui()`](https://benjaminhlina.github.io/glatar-app/reference/register_ui.md).

## Usage

``` r
register_server(id)
```

## Arguments

- id:

  the shiny namespace id in this case it is \`"register"“.

## Value

a shiny server object that contains registration server.
